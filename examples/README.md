# `oauth2-cli` Examples

## [01 basic-use](./01%20basic-use#readme)

A basic example of how to leverage the library for immediate use, connecting it to the Canvas LMS API.

## [02 ejs-peer](./02%20ejs-peer#readme)

Install [ejs](https://www.npmjs.com/package/ejs) as a peer to provide prettier formatting of browser messages in the Authorization Code flow.

## [03 custom-ejs-templates](./03%20custom-ejs-templates#readme)

Override the default messages and formatting for browser messages in the Authorization Code flow.

## [04 control-auth-timing](./04%20control-auth-timing#readme)

Rather than waiting for an API request to trigger authorization, add a specific authorization step to your logic so that you have an opportunity to provide inline documentation, etc.

## [05 token-storage](./05%20token-storage#readme)

Persist tokens between runs by saving them to a JSON file.

## [06 qui-cli](./06%20qui-cli#readme)

Use the [qui-cli](https://www.npmjs.com/package/@oauth2-cli/qui-cli) plugin for easier command line app development.
